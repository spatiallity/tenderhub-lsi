# 📄 CV Generator - Quick Summary

## ✨ Fitur Baru

Generator CV Tenaga Ahli dengan **dual format output**:

### 📘 DOCX (Editable)
- File Microsoft Word yang bisa diedit
- Cocok untuk revisi manual
- Format tabel sesuai standar LKPP

### 📕 PDF (Print-Ready)
- File siap print/submit
- Tidak bisa diedit (final)
- Auto page break untuk konten panjang

## 🚀 Cara Pakai (3 Langkah)

1. **Buka Expert Detail** → Klik expert → Klik tombol "Generate CV" (biru, pojok kanan)
2. **Isi Form** → Navigasi tab: Data Pribadi, Pendidikan, Pengalaman, Bahasa
3. **Pilih Format & Download** → Toggle DOCX/PDF di footer → Klik "Generate"

## 📋 Apa yang Sudah Bisa

✅ Data Pribadi (12 field lengkap)  
✅ Pendidikan (multi-jenjang: S3, S2, S1, D3, SMA)  
✅ Pengalaman Kerja (auto-populate dari expert history)  
✅ Kemampuan Bahasa (multi-entry)  
✅ Output DOCX (editable)  
✅ Output PDF (print-ready)  

## 🚧 Belum Tersedia

⏳ Sertifikasi & Keahlian  
⏳ Pelatihan Non-Formal  
⏳ Organisasi Profesi  
⏳ Referensi  
⏳ Upload Foto 3x4  

## 💡 Tips

- **Untuk revisi**: Pilih DOCX, edit manual di Word
- **Untuk submit**: Pilih PDF, langsung siap kirim
- **Data pengalaman**: Sudah auto-fill dari riwayat expert
- **Format tanggal**: Otomatis convert ke format Indonesia

## 🎯 Output

File akan auto-download dengan nama:
- `CV_[Nama_Lengkap]_2025.docx`
- `CV_[Nama_Lengkap]_2025.pdf`

Format sesuai standar dokumen pengadaan pemerintah (LKPP).
