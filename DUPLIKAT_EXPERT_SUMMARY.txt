═══════════════════════════════════════════════════════════════════
  🔍 ANALISIS: Duplikasi Data Expert
═══════════════════════════════════════════════════════════════════

ROOT CAUSE DITEMUKAN:
  ❌ Function addExpert() TIDAK return value (return undefined)
  ❌ Form tidak di-reset setelah save karena if (success) = if (undefined) = false
  ❌ User bisa klik tombol "Simpan" berkali-kali
  ❌ Setiap klik = 1 expert baru di database = DUPLIKAT!

CONTOH FLOW YANG SALAH:
  1. User isi form expert
  2. User klik "Simpan"
  3. addExpert() dipanggil → API POST /experts → Expert tersimpan
  4. addExpert() return undefined (tidak ada return statement)
  5. if (success) = if (undefined) = false
  6. Form TIDAK di-reset, TIDAK ditutup
  7. User klik "Simpan" lagi (karena form masih terbuka)
  8. Expert yang sama tersimpan lagi → DUPLIKAT!

═══════════════════════════════════════════════════════════════════
  ✅ SOLUSI YANG DITERAPKAN
═══════════════════════════════════════════════════════════════════

1. FIXED addExpert() Function
   File: frontend/src/store/AppContext.jsx
   
   BEFORE (SALAH):
   ```javascript
   const addExpert = useCallback(async (draft) => {
     try {
       const res = await api.post('/experts', body);
       setExpertsRaw(prev => [...prev, res.data]);
       showToast('Tenaga ahli berhasil ditambahkan');
       // ❌ No return statement
     } catch (e) {
       // ❌ No return statement
     }
   }, [showToast]);
   ```
   
   AFTER (BENAR):
   ```javascript
   const addExpert = useCallback(async (draft) => {
     if (!draft.nama?.trim() || !draft.keahlian?.trim()) {
       showToast('Nama dan keahlian wajib diisi', 'warning');
       return false; // ✅ Return false
     }
     
     try {
       const res = await api.post('/experts', body);
       setExpertsRaw(prev => [...prev, res.data]);
       showToast('Tenaga ahli berhasil ditambahkan');
       return true; // ✅ Return success
     } catch (e) {
       console.error('[addExpert] Failed:', e);
       showToast('Gagal menambahkan expert', 'error');
       return false; // ✅ Return failure
     }
   }, [showToast]);
   ```

2. CREATED Cleanup Script
   File: cleanup_duplicate_experts.py
   
   Purpose: Hapus duplikat yang sudah ada di database
   
   Features:
   - Deteksi duplikat berdasarkan nama (case-insensitive)
   - Tampilkan expert mana yang akan dihapus
   - Minta konfirmasi sebelum delete
   - Keep expert pertama (ID terkecil)
   - Delete semua duplikat lainnya
